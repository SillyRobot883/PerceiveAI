تحميل التوثيق الكامل

مقدمة

مرحباً بك في التوثيق الرسمي لمشروع PerceiveAI، الحل المبتكر لتحويل التعليق الصوتي للمباريات الرياضية إلى لغة الإشارة باستخدام الذكاء الاصطناعي.

يهدف هذا التوثيق إلى مساعدة المطورين والمؤسسات الإعلامية على فهم التقنية وكيفية دمجها في منصاتهم لتقديم تجربة أكثر شمولية للمشاهدين من ذوي الإعاقة السمعية.

ملاحظة هامة

التوثيق الحالي هو للإصدار التجريبي (Beta v1.2) وهو قيد التطوير المستمر. يتم تحديث هذه الصفحة بانتظام مع إضافة ميزات جديدة وتحسينات.

مثال على استدعاء API

التثبيت

يمكن تثبيت مكتبة PerceiveAI بسهولة باستخدام مدير الحزم npm أو yarn. تتوفر المكتبة للعديد من بيئات التطوير.

# باستخدام npm
npm install perceive-ai --save
# باستخدام yarn
yarn add perceive-ai

متطلبات النظام

  • Node.js v14.0.0 أو أعلى
  • متصفح حديث يدعم WebGL و WebAssembly
  • اتصال انترنت مستقر (مطلوب للإصدار السحابي)
  • وحدة معالجة رسومات مُوصى بها للإصدار المحلي
